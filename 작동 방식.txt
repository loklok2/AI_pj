1. app.py: Streamlit 웹 애플리케이션의 메인 파일. 이미지 업로드, 캡션 생성, 번역 기능을 제공.
2. run_app.py: Streamlit 서버 실행 및 ngrok으로 외부 접속 터널을 실행하는 스크립트.
3. google_translate.py: Google Translate API를 사용하여 영어 캡션을 한국어로 번역.
4. image_captioner.py: 
- 이미지 처리 핵심 로직
- 기능: 배경 제거, 캡션 생성, 번역
- 불필요 정보 제거, 의류 관련 정보 추출
- 각 단계별 처리 시간 측정

1. app.py
기능: Streamlit 웹 애플리케이션으로, 사용자가 이미지를 업로드하고, 해당 이미지를 기반으로 캡션을 생성하고 번역된 결과를 제공합니다.
주요 메서드:
load_model(): ImageCaptioner 모델을 불러옵니다.
st.file_uploader(): 이미지를 업로드하는 인터페이스를 제공합니다.
caption_and_translate(): 업로드된 이미지를 기반으로 배경 제거, 캡션 생성, 번역을 수행하고 결과를 출력합니다.
2. run_app.py
기능: Streamlit 서버를 실행하고, ngrok을 사용해 외부에서 접속할 수 있도록 터널을 설정하는 스크립트입니다.
주요 메서드:
run_streamlit(): Streamlit 앱을 지정된 포트(8502)에서 실행합니다.
run_ngrok(): ngrok을 실행하여 외부 접속을 위한 터널을 생성합니다.
3. google_translate.py
기능: Google Translate API를 사용하여 영어로 생성된 캡션을 한국어로 번역하는 기능을 제공합니다.
주요 메서드:
translate_text(text, target_language='ko'): 입력된 텍스트를 Google Translate API를 사용하여 번역합니다.
4. image_captioner.py
기능: 이미지를 기반으로 배경을 제거하고, 캡션을 생성하고, 이를 한국어로 번역하는 기능을 제공합니다. 배경과 포즈와 같은 불필요한 정보는 제거하고, 옷에 관련된 정보만 남기도록 설계되었습니다.
주요 메서드:
load_image(): 이미지 파일을 불러옵니다. URL, 파일 경로, 이미지 객체를 처리할 수 있습니다.
remove_background(): rembg 라이브러리를 사용하여 이미지의 배경을 제거합니다.
generate_caption(): 이미지를 분석하여 캡션을 생성합니다.
post_process_caption(): 생성된 캡션에서 불필요한 정보를 필터링하고, 의류와 관련된 정보만 추출합니다.
translate_to_korean(): 영어로 생성된 캡션을 한국어로 번역합니다.
caption_and_translate(): 전체 흐름(이미지 불러오기 -> 배경 제거 -> 캡션 생성 -> 번역)을 처리하고, 각 단계별 처리 시간을 측정합니다.
